Версия 3.6
-----------------------------------
- Добавлена команда чтения текущего состояния всех входов INPUTS=?
Возвращает ответ в формате
IN1=0,IN2=0,IN3=0,IN4=0,ADC=20;

Версия 3.5C
-----------------------------------
- При появлении лог "0" на входе 1, эмулируется команда включения выхода 1

Версия 3.4
-----------------------------------
- Дополнительная проверка правильной инициализации состояния входов при старте

Версия 3.3
-----------------------------------
- Добавлена возможность отключения уведомления о выполнении SMS команд

Новые команды:

SMS_ACK - может принимать значение 0 (отключить уведомления) или 1 (включить уведомления)

Версия 3.2
-----------------------------------
- Исправлена ошибка с отправкой ложных срабатываний по входам

Версия 3.1
-----------------------------------
- Увеличено число входов до 4
- Добавлено отключение таймера перезагрузки
- Добавлено отключение Heartbeat SMS
- Добавлено сохранение состояния входов в энергонезависимой памяти

Новые команды:

SAVEINPUTS - может принимать значение 0 (не сохранять) или 1 (сохранять)

Версия 2.0
-----------------------------------
Добавлена задержка повторной отправки сообщений при срабатывании триггеров по входам АЦП, СК1, СК2.
Добавлен выбор типа срабатывания по входам АЦП, СК1, СК2 - фронт, спад, оба направления.
Новые параметры работы:

Выбор направления срабатывания (1 - фронт, 2 - спад, 3 - оба)
ADCFRONT
IN1FRONT
IN2FRONT

Задержка отправки посторного сообщения (в сек)
ADCDELAY
IN1DELAY
IN2DELAY

Периодическая перезагрузка
Период плановой перезагрузки (в сек)
REBOOTPERIOD

Количество плановых перезагрузок между отправкой тестового SMS (шт)
ALIVESMS

Версия 1.0
-----------------------------------

Пароль по умолчанию: 0000

Поддерживаемые SMS команды:
Управление выходами:
OUT1=0 (откличить выход)
OUT1=1 (включить выход)
OUT2=0(1)

Получение текущего значения АЦП:
ADC=?

Пароль доступа: 
PASS=Password123 (строка из латинских букв и цифр)

Белый список для управления звонком: 
WHITE=+79261234567,+79169876543,.... (перечень телефонных номеров в формате +7...., разделенных запятой, количество не ограничено)

Список номеров для оповещения об изменении состояния входов:
ALERT=+79261234567,+79169876543,.... (перечень телефонных номеров в формате +7...., разделенных запятой, количество не ограничено)

Пользовательское название входов для SMS оповещения:
(текст, только латинские буквы)
ADCTXTOVR=... превышение заданного значения
ADCTXTUND=... понижение от заданного значения
IN1ONTXT=... сообщение при лог. 1 на входе
IN1OFFTXT=... сообщение при лог. 0 на входе
IN2ONTXT=... --//--
IN2OFFTXT=... --//--

ADCVAL=5000 (пороговое значение АЦП на входе терминала, в мВ)
ADCHYST=500 (окно гистерезиса, в мВ)

Время включения выхода
OUT1TIME=10 (значение в сек)
OUT2TIME=...

Формат SMS:
PASSWORD;OUT1=1;ADCTXT=Text dlya adc;WHITE=+79161234567,+79267654321
